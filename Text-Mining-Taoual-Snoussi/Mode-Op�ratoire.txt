Mode opératoire du projet :
1- le fichier Dela contient les dictionnaires qu'on a créé pour les verbes de mouvement ainsi qu'un deuxième dictionnaire pour les noms des lieux(Placename).
2- le fichier Graphs contient les huit graphes qu'on a créé avec le graphe (Textmining)celui qui fait la reconnaissance des entités nommées (EN) et les entités nommées étendu (ENE).
3- le fichier Corpus contient corpus2017 avec lequel on a enrichie les dictionnaires et corpus2018 sur lequel on a testé.
4- le fichier CasSys contient la cascade qu'on a créé.
5- le script Python (textmining.py) pour avoir les verbes de mouvement et les entités de lieu.
 
*En utilisant le logiciel Unitex :
1- On commence par ouvrir un texte du corpus2018 puis faire le prétraitement du texte(tokenization).
2- On ouvre nos dictionnaires(verbes-mouv,placename) ensuite on les compresse pour avoir la forme fst.
3- On clique sur "Apply Lexical Ressources" qui se trouve dans "Text" et on selectionne nos dictionnaires ainsi que Dela_fr.bin.
4- Ensuite on clique sur "Local Pattern" , on choisit notre graphe (textmining.grf) et on choisit Merge with input text dans la section "Grammar Outputs".
5- On clique sur Search ensuite sur Build concordance.


*Pour utliser la cascade : 
1- On clique sur Apply CasSys Cascade et on choisit la cascade qui est déjà créee. (TM2.csc)
